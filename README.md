# DSM050 - UK Renewable Energy Planning Database: Data Visualisation Coursework

Data visualisation coursework analysing the UK Renewable Energy Planning Database (REPD, Q1 2026),
exploring three research questions:

- **RQ1 - Capacity mix over time**: how has the technology mix of UK renewable capacity evolved?
- **RQ2 - Regional distribution**: how is capacity distributed across UK regions, and how has that
  geography evolved (including offshore, which sits outside any land region)?
- **RQ3 - Policy scheme effects**: how do support schemes (RO, FiT, CfD) relate to project scale?

The full write-up, with figures and discussion, is in [`submission/report.pdf`](submission/report.pdf).

## Reproducing the analysis

1. Create a virtual environment and install the pinned dependencies:
   ```
   python -m venv dv_venv
   dv_venv\Scripts\pip install -r requirements.txt
   ```
2. Run `submission/DSM050_Renewables.ipynb` top to bottom. This regenerates every file in
   `submission/figures/`.
3. Compile the report from `submission/report.tex`:
   ```
   pdflatex report.tex
   bibtex report
   pdflatex report.tex
   pdflatex report.tex
   ```
